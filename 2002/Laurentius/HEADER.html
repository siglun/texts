On the S:t Laurentius Digital manuscript library

  • The search engine documented. Read the paper. It is using metafont and dvi, so the typography doesn't look that good on screen, but is nice in print.
  • The XML source laurentius.xml is rendered automatically as html. To see it, use ./laurentius.xml?passthru=1. Please note that there is a dirty hack in this, which makes the xml invalid -- the entities &mdash; and &ndash; They are recasted as empty elements, e.g., <mdash/> such that I can catch them using xslt. Cf ./SCRIPTS/tei_to_roff.xsl and my default tei to html style sheet.
  • A worked example for printing TEI XML using groff. Download laurentius.tar.gz
