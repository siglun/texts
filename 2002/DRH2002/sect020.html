3. XML: Documents and Data

  • XML in two domains -- Document Like Objects, DLO
    • Document domain (i.e., "encoded text" -- prose,poetry)
      • HTML
      • DocBook
      • TEI
    • Data domain -- Record Like Objects, RLO
      • Structured data. Easily created from a RDBMS and likewise easily ingested into such a database.
        • Purchase order
  • Master is document domain.
    • What a Manuscript librarian/researcher expects in a printed catalog
    • Assigns metadata to volumes.
    • Possible to describe objects with arbitrary granularity. That is, we may assign metadata to ranges of text, eg., ff33r:10-37v:15
