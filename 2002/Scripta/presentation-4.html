4. Current developments

  • About 1800 theses classified in CERIF
    • A browsing structure for Scripta
  • Improve editorial procedures for data sheets
  • Convert current database
    • A XML DTD for document data sheets. Sample record.
    • Ensure coexistence of legacy metadata and metadata produced within the new infrastructure.
    • XML Style Sheet Transformations (XSLT) technology used for various tasks, including brief and detailed hit presentations, editing, on the fly generation of data sheets for printing etc.
  • Bibliographic search including support for
    • US MARC/MARC 21 record syntax
    • Z39.50 cross searching capabilities with library OPACS and SAFARI.
    • Methods devised for limiting searches to dissertations while cross searching SAFARI, Libris and BTJ Burk.
  • Operational infrastructure for the archive
