Introduction

These statistics were collected 2000-05-02 from the following sources, SAFARI [1] and SiL [2]. The two services are based on identical methods, and use the same metadata element set. The two databases contain 7775 and 1217 records, respectively.

Below I have tabulated

  • Target audience by domain
  • Resource category by domain
  • Resource category by audience
Where the mapping between category and resource type is defined in Appendix A.

Appendix A: Mapping between resource type and category

my %typemap = ('bibliographic' => ['text.thesis',
                                   'text.thesis.doctoral',
                                   'text.thesis.masters',
                                   'text.techreport',
                                   'text.preprint',
                                   ],

               'expertise'     => ['text.x-expertise.organi[sz]ational',
                                   'text.x-expertise.personal',
                                   'text.x-expertise',
                                   ],

               'pressrelease'  => ['text.correspondence.x-pressrelease'],

               'agent'         => ['text.homepage',
                                   'text.abstract.x-project',
                                   'text.homepage.x-project',
                                   'text.homepage.organizational',
                                   'text.homepage.personal',
                                   ],

	       'invalid'       => ['invalid',
                                   '(thesis)|(avhandling)',
                                   'ex[\.\s]*?arb', 
                                   'uppsats', 
                                   '(report)|(rapport)',
                                   'examensarbete',
				   ],

               'other'         => ['.*',
                                   'text.article',
                                   ],
               );
