About the web site

SVEP lives on balder.lub.lu.se:/localdisk1/svep-project. The service has one virtual server on that machine, www.svep-project.se Most content on the site are maintained by project partners using the Keystone Software.

There is one exception; in addition to the public site the project also has a password protected intranet. Parts of that intranet is maintained using an upload script which permits partners to upload internal information. The intranet is on http://www.svep-projekt.se/internt/. The password protection is a very simple one, with a .htaccess file in the protected directory. In addition the project has a mailing list maintained by majordomo@munin.lub.lu.se with a standard web archive.

The service is based on the Biblioteca1 service distributed by Indexdata. See Keystone Hotel
